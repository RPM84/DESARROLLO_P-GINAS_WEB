COMANDOS BÁSICOS GIT BASH

cd
Comando para cambiar de carpeta
ls
Comando para mostrar todos los elementos de la carpeta actual
mkdir
Comando para crear una carpeta nueva
pwd
Comando para indicar la carpeta actual
cp
Comando para copiar archivos o carpetas
mv
Comando para mover archivos o carpetas
rm
Comando para eliminar archivos (rm-r)

COMANDOS BÁSICOS GIT HUB

git config --global user.name "xxxxx"
Configurar Nombre que salen en los commits
git config --global user.email xxxxx1@gmail.com
Configurar dirección de e-mail
git init
Iniciar GIT en la carpeta donde esta el proyecto
git clone "url"
Clonar el repositorio de github
git add *
Añadir todos los archivos para el commit
git commit -m "Texto que identifique por que se hizo el commit"
Comando para ejecutar el commit
git push
Comando para subir el repositorio a la nube
